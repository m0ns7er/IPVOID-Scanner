# IPVOID-Scanner
This tool will help you check if a list of IP's have been blacklisted. 

## Installtion:
    $ git clone https://github.com/m0ns7er/IPVOID-Scanner.git
    $ cd IPVOID-Scanner
    $ pip3 install -r requirements.txt

## Usage:
    $ python3 black.py {list}

## Requirments: 
* python3
* http.client
* re
* colorama
